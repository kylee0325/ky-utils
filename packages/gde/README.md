# gde-packages

## 简介

稿定设计企业解决方案基础通用能力的集合库。

## 特性

- 基于 `vue2`、`@gaoding/gd-antd`、`@vue/composition-api`。
- 提炼各项目开发过程中可复用的通用能力。
- 目前包含三个包： `@gde/ui`、`@gde/utils`、`@gde/compositions`。

## 准备开始

```bash
yarn

// 开发
yarn dev

// 构建
yarn build
```

## 发包

各个包独立发版

```bash
# 进入对应的包内
cd packages/*

# 发包
npm publish
```
